REAPER API 中文文档
=============================================

欢迎！这里是非官方的 REAPER API 中文文档。希望提供本地化的 REAPER API 说明，以及提供更详细的范例。

你可以在 `Github <https://github.com/wastee/REAPER-API-Document-CN>`_ 上参与该文档的工作，也可以在 `REAPER API 填写表 <https://docs.qq.com/form/page/DS1dRbkFSeXZGVXha>`_ 提交更多 API。填写表非常简单，最麻烦的清洗整合由脚本自动完成。期待你的参与！

本项目主要参考：

https://www.reaper.fm/sdk/reascript/reascripthelp.html
https://mespotin.uber.space/Ultraschall/Reaper_Api_Documentation.html
https://www.extremraym.com/cloud/reascript-doc
