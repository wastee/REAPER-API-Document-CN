``APIExists``\(*function_name*)
   检查你提供的名字是否存在于REAPER API

      :参数:
         **function_name**
            字符串，你提供的一个函数名字

      :返回:
            窗口实例

   
         
   .. rubric:: 范例
   .. code-block:: python
      :caption: Python:

         RPR_APIExists(function_name)
